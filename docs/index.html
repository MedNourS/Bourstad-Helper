Bourstad Helper Documentation

Welcome

Welcome to the Bourstad Helper documentation. This tool assists users in analyzing and managing stocks from the Bourstad platform.

Features

  • Real-Time Stock Data
  • Historical Price Charts
  • Stock Recommendations
  • Interactive Dashboard

Setup

To get started, follow the steps below:

  1. Clone the repository: git clone https://github.com/your-username/Bourstad-Helper.git
  2. Install dependencies: pip install -r requirements.txt
  3. Run the Streamlit app locally: streamlit run bourstad/dashboard.py

Alternatively, you can access the hosted dashboard directly using the link below.
