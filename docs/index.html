Bourstad Helper Documentation

Welcome

Welcome to the Bourstad Helper documentation. This tool assists users in analyzing and managing stocks from the Bourstad platform.

Features

  • Real-Time Stock Data
  • Historical Price Charts
  • Stock Recommendations
  • Interactive Dashboard

Setup

To get started, follow the steps below:

  1. Ensure you have Python 3.8 or higher installed on your system.
  2. Clone the repository: git clone https://github.com/your-username/Bourstad-Helper.git
  3. Navigate to the project directory: cd Bourstad-Helper
  4. Create a virtual environment: python -m venv env
  5. Activate the virtual environment:
    • On Windows: env\Scripts\activate
    • On macOS/Linux: source env/bin/activate
  6. Install dependencies: pip install -r requirements.txt
  7. Run the Streamlit app locally: streamlit run bourstad/dashboard.py

Alternatively, you can host the dashboard using GitHub Pages and port forwarding:

  1. Ensure the repository is public on GitHub.
  2. Push your changes to the gh-pages branch:
    • Install the ghp-import tool: pip install ghp-import
    • Build the static files for the dashboard: streamlit run bourstad/dashboard.py --server.port 8501
    • Run ghp-import -n -p -f dashboard_output_directory to deploy to GitHub Pages.
  3. Enable GitHub Pages in the repository settings and set the source to the gh-pages branch.
  4. Access the dashboard at https://your-username.github.io/Bourstad-Helper.

Troubleshooting

If you encounter issues during setup, consider the following:

  • Ensure all dependencies are installed correctly by running pip list.
  • Check if Streamlit is installed by running streamlit --version.
  • Verify that your Python version is compatible (3.8 or higher).
  • Consult the Streamlit documentation for additional help.
